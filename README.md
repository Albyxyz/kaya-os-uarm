# #############################################################################	#
#                                   README                                     	#
# #############################################################################	#
#                                                                              	#
#	Decomprimere il progetto usando il comando:                             #
#										#
#		$ tar -xvf kaya2014.tar.gz                                 	#
#										#
#   ------------------------------------------------------------------------   	#
#										#
#	Per generare il file eseguibile spostarsi nella cartella kaya2014  	#
#	appena creata e lanciare il comando:	   				#
#										#
#		$ make								#
#										#
#   ------------------------------------------------------------------------   	#
#									   	#
#	Per testare (p1test) il progetto eseguire l'emulatore del processore 	#
#	uarm con il comando:							#
#										#
#		$ uarm								#
#										#
#	Dalle impostazioni nella sezione "Boot" clickare su Browse... e 	#
#	selezionare il file kaya.core.uarm					#
#										#
#	Clickare il pulsante "Power" per accendere l'emulatore.			#
#	Da Terminals aprire un terminale.					#
#	Clickare il pulsante "Play" e verificare l'output.			#
#										#
# ############################################################################# #
#				PULIZIA						#
# ############################################################################# #
#										#
#	Per eliminare i file non necessari per l'esecuzione eseguire il		#
#	comando:								#
#										#
#		$ make clean							#
#										#
#   ------------------------------------------------------------------------   	#
#										#
#	Per eliminare tutti i file generati eseguire il comando:		#
#										#
#		$ make cleanall							#
